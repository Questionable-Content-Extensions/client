Hi! While this is a React project, you're not actually meant to open this page, as there's no "React app" website here — QCEXT is a browser userscript.

If you're uncertain how to get started, here's what you do:

  1. You can leave the npm run start you (likely) started to get to this page running, as the development version of the userscript is coded to reload the latest version when it loads on the page. If you didn't, you should run npm run start and keep it running while you're developing.
  2. Run npm run build. This will create a file @ <project-root>/dist/qc-ext-dev.user.js. Add this file to your browser as a userscript.
  3. That's it. You're done. Visit questionablecontent.net and start coding.

You probably also want to run Storybook when developing. Run npm run storybook and visit http://localhost:6006/ to see the qc-ext components in isolation
